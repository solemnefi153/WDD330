Week 8 Notes

Transform

Is a property in CSS that lets you use functions to change the appearance of you elements

Transform Functions

Translate: allows you to move elements left and right, up and down. You can also use the translateX and translateY if only one direction is desired. It is important to notice that translate works only with block and inline-block elements.

Scale: This function changes the scale of an element. You can pass one or two values. Also you can use the scaleX or scaleY to specify the direction of the desired scale.

Rotation: Rotates an element clockwise based on given degrees.

Skew: It Skewes the element. It requires 2 values representing the desired skew degrees

Transitions

The transition-property property lets you define the property that you will use for the transitions of an element. We then can assign the transform property so that it defines how our transitions will behave.

You can define more details of the transitions as follows: 

            
    .myTransitionClass{ 
        transition-property: transform, 
        color; transition-duration: 0.2s, 
        0.1s; transition-timing-function: ease-out, 
        linear; transition-delay: 50ms; 
    }

Animations

To animate elements, we need to create the rules of the animation and then we attach that animation to our elements.

This is how you create an animation: 

            
    @keyframes bgMove { 
        100% { 
            background-position: 120% 0; 
        } 
    }

To attach an animation to an element we us the animation-name property.

Canvas

The canvas API lets us draw on the UI using JavaScript. A canvas should have a defined width and height. 

            
    < canvas id="myCanvas" class="myCanvas" width="200" height="200"> 
        Sorry! Your browser doesn’t support Canvas. 
    </canvas>

Using JavaScript we can store the element from the DOM into a variable 

            
    var canvas = document.getElementById("myCanvas");

Then we need to define the context. In this case the context is 2d 

            
    var context = canvas.getContext("2d");

We define the color of the line that we will use to draw. 

            
    context.strokeStyle = "red";

We then we paint a border of the figure and the fill it with a color 

            
    context.strokeRect(10, 10, 100, 100);
    context.fillStyle = "rgba(0, 0, 255, 0.5)"; context.fillRect(10, 10, 100, 100);

SVG

Just as canvas, SVG allows you to create drawings on the DOM using JavaScript.
