Week 9 Notes

The window object

Javascript enviroments have a global object. In the browser the “window” object is that global object that encapsulates all the global variables and methods.

Browser object model (BOM)

Collection of properties and methods that contain information about the browser and the computer screen.

Window Location Property

It contains information of the URL of the current page. Some properties of the location property are:

  • href
  • protocol
  • host
  • hostname
  • port
  • etc

Window History

It gives access to information about any previously visited page.

Creating pop ups

We can use the window object to open a new window. This window could be displayed a s a pop up or as an additional tab in the browser. 

            
    const popup = window.open('https://sitepoint.com','
    SitePoint','width=400,height=400,resizable=yes');
                
    popup.close();

Screen information

The window.screen property has other properties that allow us to get information about the current screen. Some properties of the screen property are: height, width, availWidth, and colorDepth.

Document Object

Each window object contains a document object. This object manages the html and text that is displayed in the browser.

Cookies

Small files that are saved in the user’s computer. Sites can create this file in the user’s computer to save user preferences and some important data that can help the user navigate the same site next time. Only the server that created the cookie can have access to that cookie from the user’s computer.

HTML 5 APIs

“data-“ attribute

This attribute allows you to create custom attributes in html elements that can be used in JavaScript to make the pages more dynamic.

Geolocation

Geolocation Is the API used to get the geographical position of the device. If the geolocation is available, it would be grabbed from the navigator object. 

            
    navigator.geolocation.getCurrentPosition(youAreHere);


    function youAreHere(position) {
        console.log(`Latitude: ${position.coords.latitude},
        Longitude: ${position.coords.longitude}`); 
    }
