How to know the needs of my customers and users?

  • Be there: Help the user with their decision-making process when navigating my software application. (Short Tutorials, Help Section? Quick solutions to the user problems and questions, Good prompts)
  • Be useful: The users need to know what my site is about. Make application intuitive
  • Be quick: Remove unnecessary steps and improve loading times

How to improve speed?

  • Reduce image usage and size
  • Minimize and compress css and Js
